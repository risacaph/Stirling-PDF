Tools & Functionality

Papyra groups its 60+ tools into categories. Browse them in the Tools panel, or search by name or synonym.

Categories

  • Signing — add and validate digital signatures, sign with a server certificate.
  • Document Security — add/remove passwords, permissions, redaction, sanitise, watermark.
  • Verification — validate signatures and compare documents.
  • Document Review — view, annotate, and read PDFs.
  • Page Formatting — rotate, crop, scale, rearrange, add page numbers, split, and merge.
  • Extraction — extract text, images, and pages; convert to and from other formats.
  • Removal — remove pages, images, metadata, or annotations.
  • Automation — pipelines and watched folders (see Pipelines & Automation).
  • General & Advanced Formatting — compression, OCR, PDF/A, edit text, and more.
  • Developer Tools — inspect and manipulate PDF internals.

The multi-tool workflow

Upload a document once, then chain tools: the output of one tool becomes the input to the next, without re-uploading. Undo an operation, preview results, or pin files to keep them active across tools.

Which tools you can run depends on your access plan. Tools outside your tier are shown greyed out.
