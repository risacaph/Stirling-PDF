System Settings

Common system.* settings and their environment variables.

SettingDefaultPurpose
SYSTEM_DEFAULTLOCALEautoForce a UI language (e.g. en-US, de-DE). Empty auto-detects from the browser.
SYSTEM_GOOGLEVISIBILITYfalsetrue serves an allow-all robots.txt; false keeps the instance out of search engines.
SYSTEM_SHOWUPDATEtrueShow when a new release is available.
SYSTEM_SHOWUPDATEONLYADMINtrueOnly admins see the update notice.
SYSTEM_ENABLEANALYTICSnullMaster analytics toggle — see Analytics & Telemetry.
SYSTEM_ENABLEMOBILESCANNERtrueQR-code mobile upload (needs SYSTEM_FRONTENDURL).
SYSTEM_MAXDPI500Max DPI for PDF-to-image conversion.

Public URLs

For anything involving external callbacks or links — SSO, invite emails, the mobile scanner — set the public URLs:

  • SYSTEM_BACKENDURL — public base URL of the backend (required for SSO).
  • SYSTEM_FRONTENDURL — public URL users click in invite emails; defaults to the backend URL.

CORS

SYSTEM_CORSALLOWEDORIGINS defaults to empty, which allows all origins with credentials — it does not disable CORS. Set explicit origins to lock it down.
