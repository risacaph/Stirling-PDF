Access Plans

Papyra removes the vendor licensing model entirely — all features are unlocked. In its place is a simple, admin-managed per-user access plan that controls how long a user's access lasts and which tools they may use.

The tiers

PlanDurationAccess
Free7 daysA trial with basic tools (merge, split, rotate, compress, page operations, image ↔ PDF).
Pro1 yearMost tools.
Ultimate5 yearsEverything, including pipelines and automation.

How it works

  • Newly created users are automatically placed on the Free 7-day trial.
  • An administrator can assign any plan to any user at any time; assigning a plan resets its expiry to now plus the plan's duration.
  • When a plan expires, the user can still sign in, but the app becomes read-only — tool operations are blocked until an admin renews the plan.
  • Tools above a user's tier appear greyed out and locked in the tool picker, with a tooltip explaining why.
  • Administrators are exempt — they always have full, unlimited access.
Enforcement happens on the server, so the plan is authoritative regardless of the UI. Users can see their own plan and expiry any time under Settings → Account.

Assigning a plan

Admins manage plans from Settings → People — see Managing Users & Teams.
